Router IP Address Configuration

 Project Overview

This project demonstrates the configuration of IP addresses on a Cisco router
using Cisco Packet Tracer.

The objective was to assign IP addresses to router interfaces, configure
network connectivity, and verify communication between connected networks.

 Technical Specifications & Tools

* **Software:** Cisco Packet Tracer v8.x
* **Hardware Simulated:** Cisco ISR 4331 Router
* **Protocols:** IPv4, ICMP
